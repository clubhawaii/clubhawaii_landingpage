설명회 신청 페이지
만들기 사용 설명서

처음이셔도 괜찮아요. 아래 순서대로 따라 하시면 약 10분이면 학부모님께 보낼 신청 링크가 완성됩니다.

전체 흐름 (5단계)
  1. 1. 구글폼 만들기·게시
  2. 2. 디자인 고르기
  3. 3. 정보 입력
  4. 4. 파일 다운로드
  5. 5. 인터넷에 올려 링크 만들기
① 먼저 준비 — 구글폼 신청서 만들기 · 게시 · 링크 복사
1
구글폼 신청서 만들기

먼저 학부모님이 신청할 구글폼 신청서를 만들어요. 인터넷 주소창에 forms.google.com 을 입력해 접속한 뒤, ‘내용 없는 양식(빈 양식)’을 눌러 새로 만듭니다. (구글 계정 로그인이 필요해요.)

  • 맨 위 ‘제목 없는 설문지’를 눌러 제목을 적어요. (예: ○○학원 2028 입시 설명회 신청)
  • 질문에 이름, 연락처 정도만 받으면 충분해요. (질문 추가는 오른쪽 ⊕ 버튼)
  • 각 질문 아래 ‘필수’를 켜두면 빈칸으로 신청하는 걸 막을 수 있어요.
질문은 간단할수록 신청률이 높아요. 이름·연락처만 받아도 충분합니다.
2
‘게시’ 버튼으로 공개하기

신청서를 다 만들었으면, 학부모님이 들어올 수 있도록 공개(게시)해야 해요. 게시를 안 하면 링크를 보내도 학부모님이 못 들어가요.

  • 편집 화면 우측 상단의 보라색 ‘게시’ 버튼을 누릅니다.
  • ‘응답자’가 ‘링크가 있는 모든 사용자’로 되어 있는지 확인하고 ‘게시’를 누릅니다.
꼭 확인! 응답자가 ‘링크가 있는 모든 사용자’로 되어 있어야 해요. 제한돼 있으면 학부모님이 “접근 권한이 없습니다” 화면을 보게 돼요.
3
응답자 링크 복사하기

게시한 뒤, 학부모님께 보낼 응답자 링크를 복사해요.

  • 우측 상단의 ‘게시됨’(또는 🔗 모양) 버튼을 누릅니다.
  • ‘응답자 링크 복사’를 누르면 주소가 복사돼요. (예: https://forms.gle/xxxxx)
이 링크는 잠시 뒤 생성기 STEP 2의 ‘구글폼 신청 링크’ 칸에 붙여넣을 거예요. 메모장이나 카톡 ‘나에게 보내기’에 잠깐 저장해두면 편해요.
② 생성기에서 페이지 만들기
4
디자인 고르기 (STEP 1)

생성기 링크를 열면 6가지 디자인이 보여요. 마음에 드는 디자인을 누르세요. 각 디자인 아래 ‘미리보기’를 누르면 실제 모습을 새 창으로 미리 볼 수 있어요.

디자인은 색과 글씨체만 다를 뿐, 들어가는 내용은 똑같아요. 우리 학원 분위기에 맞는 걸 고르시면 됩니다.
▲ 실제 생성기 STEP 1 화면 — 디자인을 누르면 선택돼요
5
학원 정보 입력 (STEP 2)

꼭 입력할 3가지만 채우면 페이지가 완성돼요.

  • 학원명 — 우리 학원 이름
  • 설명회 일시 — 설명회 날짜와 시간
  • 구글폼 신청 링크 — 1번에서 복사해 둔 주소를 붙여넣기

좌석·정원·전화번호 같은 추가 정보는 ‘추가 정보(선택)’를 눌러 펼쳐서 넣으면 되고, 안 넣으면 페이지에 안 나와요.

▲ 실제 생성기 STEP 2 화면 — 별표(*) 3개만 꼭 입력하세요
6
안내사항 확인 (STEP 3)

설명회 안내사항에는 예시 문구가 미리 적혀 있어요(연한 회색 글씨). 예시를 참고하시되, 우리 학원에 맞게 꼭 확인하고 수정해 주세요.

바꾸고 싶을 때만 ‘큰 제목’을 우리 학원에 맞게 고치고, 더 손보고 싶으면 ‘더 자세히 고치고 싶다면’ 박스를 열면 돼요.

바쁘시면 STEP 3은 건너뛰어도 괜찮아요. 예시 그대로 충분히 보기 좋게 나옵니다.
7
미리보기 → 다운로드 (STEP 5)

‘미리보기’로 완성된 모습을 한 번 확인하고, 괜찮으면 ‘완성 파일 다운로드’를 누르세요. HTML 파일 하나가 컴퓨터에 저장돼요.

이 파일을 그냥 카톡으로 보내면 학부모님이 못 열어요. 아래 ③단계처럼 인터넷에 올려야 링크가 생깁니다.
③ 인터넷에 올려 링크 만들기 (가장 중요!)
⚠️ 무료 가입은 꼭 하세요! 가입하지 않으면 만든 링크가 1시간 뒤에 자동으로 삭제돼요. 설명회는 보통 한 달 정도 열어두니, 구글 계정으로 1분이면 되는 무료 가입을 꼭 하세요. 가입 버튼은 파일을 올린 뒤에 나옵니다.
8
사이트 접속 후 파일 올리기
  • 인터넷 주소창에 app.netlify.com/drop 을 입력해 접속해요.
  • 회색 점선 박스가 나오면, 다운로드한 HTML 파일을 그 안으로 끌어다 놓아요(드래그).
  • 잠깐 기다리면 ○○○.netlify.app 주소가 생깁니다.
⚠️ 이 단계는 꼭 컴퓨터(PC)에서 하세요. 휴대폰에서는 파일을 끌어다 놓는 것이 되지 않아 링크를 만들 수 없어요.
9
영어 팝업이 뜨면? (당황 금지!)

파일을 올리면 아래 같은 영어 창이 떠요. 왼쪽 파란 버튼(Rename and deploy)을 누르세요. 그래야 페이지가 제대로 열려요.

⬆️ 이 파란 버튼 (Rename and deploy)을 누르세요!
오른쪽 회색 버튼(Deploy without renaming)을 누르면 페이지가 안 열릴 수 있어요. 꼭 왼쪽 파란 버튼!
10
무료 가입하고 링크 보내기
  • 주소가 생긴 화면에 뜨는 ‘Sign up(가입)’ 또는 ‘Claim’ 버튼을 눌러요.
  • 구글 계정으로 가입하면 1분이면 끝! (이래야 링크가 안 사라져요.)
  • 완성된 ○○○.netlify.app 주소를 복사해 학부모님께 카톡·문자로 보내면 끝이에요!
자주 묻는 질문
Q. Netlify 가입이 필수인가요?▶
네, 꼭 하세요. 가입하지 않으면 만든 링크가 1시간 뒤에 사라져요. 구글 계정으로 1분이면 가입돼요.
Q. 설명회 날짜가 바뀌었어요. 날짜를 어떻게 수정하나요?▶
생성기에서 내용을 고쳐 다시 다운로드한 뒤, 같은 자리(app.netlify.com/drop)에 다시 끌어다 놓으면 돼요. 가입돼 있으면 같은 주소로 업데이트됩니다.
Q. Netlify로 만든 링크 주소를 바꿀 수 있나요? (○○○ 부분)▶
네, 가입 후 사이트 설정(Site settings)에서 이름을 바꿀 수 있어요.
Q. 휴대폰으로 링크를 만들 수 있나요?▶
아니요, 꼭 컴퓨터(PC)에서 해주세요. 파일을 끌어다 놓는 과정이 휴대폰에서는 제대로 되지 않아, 링크 만들기가 안 됩니다. 노트북이나 데스크탑에서 진행해 주세요.
Q. 비용이 드나요?▶
전부 무료예요. 생성기도, Netlify 호스팅도 무료입니다.
막히는 부분이 있으면 클럽하와이로 문의해 주세요.
