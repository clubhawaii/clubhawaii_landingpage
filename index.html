2028 대입 시행계획 설명회
신청 페이지 만들기

우리 학원 2028 대입 시행계획 설명회 신청 페이지를 코딩 없이 만드세요. 디자인을 고르고 학원 정보만 넣으면 바로 완성됩니다. 본문은 예시 문구가 채워져 있으니, 우리 학원에 맞게 고치거나 그대로 쓰셔도 됩니다. 비워둔 항목은 자동으로 빠집니다.

STEP 1. 마음에 드는 디자인을 고르세요
STEP 2. 학원 기본 정보
구글폼에서 신청서를 만든 뒤 보내기 → 링크 → URL 복사를 눌러 붙여넣으세요. * 항목은 꼭 입력해야 해요.
STEP 3. 설명회 안내 글 (그대로 두셔도 됩니다)
아래는 2028 대입 시행계획 설명회에 맞춘 예시 글이에요. 연한 회색 글씨가 예시이고, 그대로 두면 그 내용이 그대로 들어갑니다. 우리 학원에 맞게 바꾸고 싶은 칸에만 직접 입력하세요. 필요 없는 섹션은 제목 옆 ‘이 섹션 빼기’를 체크하면 페이지에서 사라집니다.
‘예시를 입력칸에 채우기’를 누르면 예시 글이 입력칸에 실제로 들어와서, 한 글자씩 고치기 편해요. ‘입력 내용 비우기’를 누르면 다시 연한 예시 상태로 돌아갑니다.
STEP 4. 확인하고 내려받기
받은 HTML 파일을 Netlify Drop(app.netlify.com/drop) 같은 무료 호스팅에 올리면 학부모님께 보낼 주소가 생깁니다.
입력한 내용은 이 기기 안에서만 처리되며 어디에도 저장·전송되지 않습니다.
